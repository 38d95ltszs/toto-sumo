# トト相撲 ONLINE

大相撲の取組を予想して「持ち米」を賭ける、ブラウザだけで遊べる無料の相撲予想ゲームです。サーバー不要・単一HTMLファイルで動作し、[GitHub Pages](https://pages.github.com/) にそのまま公開できます。

## 特徴

- **実データ連動**: GitHub Actionsが毎日スポーツナビの本場所結果を自動取得。取組・勝敗・決まり手・優勝力士・三賞まで、実際の大相撲と同じ結果で遊べます（データが取れない場合は自動でシミュレーションモードにフォールバック）
- **サーバー不要の共有ランキング**: 取組結果は日付から決まるシード乱数で全プレイヤー共通に生成されるため、賭けの精算だけをSupabaseで共有すれば、友達同士で同じ番付・同じランキングを争えます
- **ログインID・パスワード認証**: 実在するメールアドレスは一切収集しません。パスワードはSupabase Auth側でハッシュ化され、運営者を含め誰も元のパスワードを見ることができない設計です（詳細は[ログイン認証仕様.md](./ログイン認証仕様.md)）
- **クロスデバイス同期**: ログインID・パスワードを設定すれば、持ち米・称号・推し力士・的中履歴・実績・未精算ベットまで、別の端末からも続きから遊べます
- **番付システム**: 千秋楽の成績に応じて幕内〜三役まで番付が上下。番付発表は本場所の暦に合わせて千秋楽の3日後(水曜10:00)に一斉に行われます
- **称号・実績**: 95種類の称号を収集。称号は四股名の前に表示され、番付表でタップすると「ひとこと」も見られます
- **相撲クイズ・ちゃんこボーナス**: 1日1問の相撲クイズ(全45問)と、毎日1回のちゃんこボーナスで無料で持ち米を増やせます
- **多彩な賭け方**: 通常の勝敗予想に加え、決まり手予想、東西対抗戦、優勝予想、三賞予想（殊勲賞・敢闘賞・技能賞）に対応
- **不戦勝・不戦敗の自動返還**: 力士の休場による不戦勝・不戦敗は的中/不的中の対象外とし、賭け金を全額自動返還します
- **過去のトト番付アーカイブ**: 終了した場所の番付・成績をいつでも振り返れます

## 遊び方

1. 入門の儀で四股名・所属一門・推し力士を決める
2. 「取組予想」タブで各取組に持ち米を賭ける（締切は取組ごとに個別）
3. 「場所予想」タブで優勝力士・三賞を予想
4. 結果は毎日自動精算。ログインすればまとめてポップアップで確認できます
5. 千秋楽まで生き残った持ち米の量と的中率で、次の場所の番付が決まります

## 技術構成

| ファイル | 役割 |
|---|---|
| `index.html` | ゲーム本体（HTML/CSS/JSを1ファイルに集約） |
| `scraper/fetch_basho.mjs` | 本場所の取組・結果・優勝/三賞をスポーツナビから取得するスクレイパー |
| `.github/workflows/update-data.yml` | 上記スクレイパーを1日3回自動実行するGitHub Actions |
| `data/basho.json` | スクレイパーが生成する、その場所の実データ |

バックエンドはSupabase（Auth + Postgres）のみ。取組のシミュレーションや判定ロジックはすべてクライアント側の決定論的な処理（シード乱数）で完結しており、専用サーバーは必要ありません。
